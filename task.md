 ## 🎯Створення фотогалереї на CSS Grid  
## Умова завдання:  

# Створи вебсторінку з фотогалереєю, яка відповідає таким вимогам:  

# Галерея повинна містити 12 зображень.  
Сітка має складатися з:  
4 колонок  
3 рядків  
Використай CSS Grid для побудови структури.  
Максимальний розмір кожного зображення:  
ширина: 300px  
висота: 300px  
Зображення повинні:  
не виходити за межі своїх комірок, масштабуватись пропорційно (без спотворення)  
Додай відступи між елементами (gap).  
Галерея повинна бути центрована на сторінці.  
## 🛠 Технічні вимоги:  
Використати властивості:  
display: grid  
grid-template-columns  
grid-template-rows  
gap  
# Для зображень застосувати:  
max-width  
max-height  
object-fit  
## 💡 Додаткове завдання (за бажанням):  
Зроби так, щоб при наведенні:  
зображення трохи збільшувалося (transform: scale)  
Додай тінь або рамку для фото.  
# Спробуй реалізувати варіант:  
одна велика картинка займає 2 колонки і 2 рядки  
# 📌 Очікуваний результат:  
Акуратна фотогалерея у вигляді сітки 4×3, де всі зображення однакового максимального розміру та гарно вирівняні.
